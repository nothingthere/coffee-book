# 对象和数组

对象和数组字面量的书写方式与JavaScript极相似。不过：

1.  如果数组中的原子或对象中的键值对单独成行，可省略`,`
2.  使用缩进而非`{}`书写对象字面量

```coffee
song = ["do", "re", "mi", "fa", "so"]

singers = {Jagger: "Rock", Elvis: "Roll"}

bitlist = [
    1,0,1
    0,0,1
    1,1,0
]

kids =
    brother:
        name: "Max"
        age: 11
    sister:
        name: "Ida"
        age: 9
```

转译为：

```js
var bitlist, kids, singers, song;

song = ["do", "re", "mi", "fa", "so"];

singers = {
	Jagger: "Rock",
	Elvis: "Roll"
};

bitlist = [1, 0, 1, 0, 0, 1, 1, 1, 0];

kids = {
	brother: {
		name: "Max",
		age: 11
	},
	sister: {
		name: "Ida",
		age: 9
	}
};
```

JavaScript中不能使用关键字作为对象属性名（除非添加引号），而CoffeeScript中则可：

```coffee
$('.account').attr class: 'attr'

console.log object.class
```

转译为：

```js
$('.account').attr({
	"class": 'attr'
});

console.log(object["class"]);
```
